### THIS IS STILL IN BETA


This gem is a Ruby wrapper for the TranscribeMe SOAP API, built on Savon

## Installation

Add this line to your application's Gemfile:

    gem 'transcribeme'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install transcribeme
